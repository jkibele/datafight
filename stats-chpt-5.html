What's this all about?¶

I'm working on getting a better grasp of some statistical concepts that I need for my research. I'm using Experimental Design and Data Analysis for Biologists (referred to as 'the book' from here on). At the same time, I want to learn how to perform the analyses I'm learning about using open source Python. This is the IPython Notebook I put together for some of the topics in Chapter 5 of the book. If you have a bit of familiarity with Python, this should help you figure out some statistics. ...maybe...

Using stats.linregress is better because it returns more information. Here's the list of returns from the SciPy Documentation:

slope : float
    slope of the regression line
intercept : float
    intercept of the regression line
r-value : float
    correlation coefficient
p-value : float
    two-sided p-value for a hypothesis test whose null hypothesis is
    that the slope is zero.
stderr : float
    Standard error of the estimate

Now I'm going to try to come up with a function for making plots like Figure 5.17 from the book.

In [20]: 
def fancy_plot(xarr,yarr,xlabel,ylabel):
    # generate all the OLS crap
    X = sm.add_constant( xarr, prepend=True )
    mod = sm.OLS(yarr, X)
    res = mod.fit()
    
    # generate predicted values for residual plot
    # first get slope and intercept from results
    intercept, slope = res.params
    # define a function based on my regression
    f = lambda x: intercept + slope * x
    # generate predicted values for my values of X
    predicted = f( xarr )
    
    # produce smoother line
    # note that the x and y values are supplied in 
    # reversed order from what I'd expect.
    smoo = sm.nonparametric.lowess(yarr,xarr,frac=0.5,return_sorted=False)
    # FYI, that works in statsmodels version 0.5+ but not in 0.4.3.
    
    figure(figsize=[15,8])
    gs = gspec.GridSpec(2,3,width_ratios=[4.5,1,4.5],height_ratios=[1,4])
    
    gs0 = subplot(gs[0])
    boxplot(xarr,vert=False)
    axis('off')
    subplot(gs[3])
    scat = scatter(xarr,yarr,marker='o',facecolors='none',s=50)
    scat.axes.plot(xarr,smoo)
    scat.axes.set_ylabel(ylabel) #('Number of Species')
    scat.axes.set_xlabel(xlabel) #('Clump Area ($m^2$)')
    gs5 = subplot(gs[4])
    boxplot(yarr)
    # The axes weren't staying lined up so I'm going to explicitly size them
    gs0.axes.set_xlim( scat.axes.get_xlim() )
    gs5.axes.set_ylim( scat.axes.get_ylim() )
    axis('off')
    
    subplot(gs[5])
    smoo2 = sm.nonparametric.lowess(res.resid,predicted,frac=0.5,return_sorted=False)
    scat_resid = scatter(predicted,res.resid,marker='o',facecolors='none',s=50)
    scat_resid.axes.plot(predicted,smoo2)
    scat_resid.axes.set_ylabel('Residual')
    scat_resid.axes.set_xlabel('Predicted Value')
